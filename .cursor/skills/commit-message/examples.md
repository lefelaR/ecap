# Commit message examples

Format: `{taskType}#{number}-{branchName}-{details}`

## issue

```
issue#7-7-mapbox-fix mapbox token not loading on public report page
```

```
issue#4-4-connect-ecap-database-dynamo-db-fix authority list empty after seed
```

## feature

```
feature#7-7-mapbox-add draggable location picker to public report flow
```

```
feature#12-12-auth-add session refresh on authority dashboard
```

## enhancement

```
enhancement#7-7-mapbox-add frontend nvmrc for lts/krypton
```

```
enhancement#4-4-connect-ecap-database-dynamo-db-move dynamodb tables to cloudformation file
```
